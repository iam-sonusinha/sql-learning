## How to use
1. Clone: `git clone https://github.com/<your-user>/sql-learning.git`
2. If using SQLite: run `./tools/init_sqlite.sh` to create example DB.
3. Open notebooks in `notebooks/`, or run `.sql` files against your DB.

## What’s inside
- `sql-basics/` — step-by-step concepts with examples.
- `projects/` — real mini-projects with dataset, queries, and results.
- `challenges/` — small practice tasks with increasing difficulty.
- `data/` — small sample datasets for practice.
